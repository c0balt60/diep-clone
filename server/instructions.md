# Diep-Clone Backend

This diep.io clone uses Python with FastApi for backend.

## Installation

(optional) After cloning the repo, create a virtual environment by running `python -m venv .venv`.

(optional) To activate the virtual env run  `.venv/Scripts/activate` or any one of the activation scripts.

(required) To install required packages, run `pip install -r requirements.txt`

To run the program, run `python app/main.py`
